    You love Ru Paul's Drag Race?! So do I! This is an app that allows users to generate makeup suggestions and inspirational quotes from a 'Queen'.   • YBS Galactic Tours: Mars Edition | Team Programming

    Ever wondered what it's like on Mars? This is a virtual space touring app, that allows users to choose a location on Mars to virtually explore. At YBS Galactic, user-safety is paramount, so the user is shown only upcoming tour dates that are free from asteroids! Built with React, two of Nasa's REST APIs, and group programming. This application is a work in progress, but feel free to see what it looks like so far. This project was built with three other great developers: Caroline Manalo, Dianna Plaza, Moriah Vice.
